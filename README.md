## Installation and Usage

### Requirements:

* Node.js installed

### Steps:
1. Clone repo on your local machine:
```
$ git clone https://github.com/mkavo92/Bloc-jams-React
```
2. Install dependencies:
```
$ cd Bloc-jams-React
$ npm install
```
3. Execute the app:<br/>
```
$ npm start
```

4. App now running on localhost:3000
